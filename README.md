# מתפרקים — אתר סטטי לפריסה ב-Vercel

## מבנה הקבצים
```
vercel/
  index.html           ← דף הבית
  webinar-landing.html ← דף הנחיתה לסדנה
  thank-you.html       ← דף תודה אחרי הרשמה
  README.md            ← קובץ זה
```

## איך מעלים ל-GitHub

1. היכנסי ל-github.com וצרי ריפוזיטורי חדש
2. שם מומלץ: `mitparkim-site`
3. בחרי **Public** (בחינם)
4. העלי את הקבצים: index.html, webinar-landing.html, thank-you.html

## איך מחברים ל-Vercel

1. היכנסי ל-vercel.com
2. לחצי **Add New Project**
3. בחרי את הריפוזיטורי `mitparkim-site`
4. **Build Command**: השאירי ריק
5. **Output Directory**: השאירי ריק (או `.`)
6. לחצי **Deploy**

## כתובת האתר

תקבלי כתובת בסגנון:
`https://mitparkim-site.vercel.app`

## איך מעדכנים בעתיד

1. בקשי עדכון מהמעצב
2. קבלי קבצים מעודכנים
3. החליפי את הקבצים בריפוזיטורי ב-GitHub
4. Vercel יפרוס אוטומטית תוך שניות
